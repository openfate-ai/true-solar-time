Why this package exists

Standard clock time is a legal convention. True Solar Time depends on longitude, equation-of-time drift, and timezone behavior that most apps smooth over or ignore. This package exists for teams that need reproducible astronomical correction inside real software.

Meeus-grade correction

Uses astronomical algorithms to account for the equation of time instead of relying on rough fixed-offset shortcuts.

Timezone edge cases included

Handles IANA timezones, DST gaps and overlaps, and historical offset shifts in a deterministic way.

Built for downstream engines

Feed accurate corrected time into Bazi, astrology, calendrical, or scheduling logic without rewriting the time layer every time.
